# Rotinas — Cérebro Base

## Primeira abertura

1. Retomar a tarefa que já veio no pedido.
2. Iniciar o relógio local do ciclo.
3. Entender uma situação recorrente sem aplicar formulário.
4. Pedir o menor caso real e a autorização necessária.
5. Transformar, ajustar, aprovar e salvar.
6. Produzir uma segunda saída usando só o contexto salvo.
7. Fechar com recibo e aprendizado visível.

## Próximas aberturas

Começar pelo trabalho atual. Consultar o contexto já aprovado antes de pedir novamente a mesma
explicação. Uma nova fonte amplia o cérebro; não substitui o que já foi validado sem decisão humana.

## Lote concierge

Congelar a versão antes do primeiro convite. Registrar T0–T4 pelo script do produto e comparar as
três ativações somente depois que todas terminarem.
