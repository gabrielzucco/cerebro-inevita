# Vaibhav Gupta (AI Experience) — slop é código que você não lê
> Combater código ruim de IA não é gerar menos — é ler menos e melhor.

## As sacadas
- **A definição de slop.** *"Slop is just any code you don't read."* `[00:25]`
- **Menos slop hoje do que nunca — aproveite.** *"This is the least amount of slop that your codebase will ever have. Cherish it."* `[00:26]`
- **Padrão não se impõe, se materializa num arquivo.** *"Some of them want to use Claude, some of them want to use Codex... So instead of trying to hold standards in our code base, we built an architecture.md file. This file has to be incredibly small, and it can only have things that will not change for months or for years."* `[00:54]`
- **Escrita é o que fica; código é descartável.** *"Code can be swapped, writing cannot."* `[01:41]`
- **Qualidade só sobe quando alguém é obrigado a ler.** *"This last rule was, if you're going to ship a design doc, to require people to actually go read it. And with this last standard, we suddenly had design docs that were incredibly high quality."* `[02:45]`
- **O que separa resultado de lixo é a curadoria humana.** *"We can have humans collaborate with these issues to figure out which ones are real, which ones are hallucinations, which ones don't have taste."* `[04:35]`
- **Não ler tudo, escolher o que ler.** *"Instead of having to understand all the code, I can opt into what parts of the code I want to read and understand, go to the exact lines when I really care about them."* `[09:21]`
- **Código é confiança — e por isso a gente ainda não confia na IA.** *"Code is a matter of trust. The reason that we don't use LLM code blindly is because we don't trust it yet, because the systems underneath them don't have enough rigidity."* `[15:43]`

## O que prova pra INEVITA
Slop = "qualquer código que você não lê" é a mesma lei do contexto cru: despejar bruto não-curado na IA gera lixo, porque ninguém leu. O que vira resultado é a curadoria — ler o que a IA produz, decidir o que tem "taste" e o que é alucinação. Mais contexto não é melhor; contexto certo, lido e verificado, é. Por isso o cérebro opera átomos citados, nunca o bruto inteiro.
