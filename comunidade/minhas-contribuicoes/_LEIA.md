# Minhas contribuições

Esta área é tua → comunidade. O cérebro pode sugerir uma contribuição; não pode doar teu contexto.

Fluxo: `propostas/` → tua aprovação do texto exato → `aprovadas/` → teu consentimento de envio
→ `enviadas/`.

**Dois consentimentos:** aprovar o payload não é enviar. Nenhum passo manda bruto, PII, segredo,
decisão privada ou output identificável.
